How to Set Up Google Sheets Integration
  1. Create a Google Sheet with room prices
  2. Enable Google Sheets API in Google Cloud Console
  3. Create an API key (no authentication required for public sheets)
  4. Share your Google Sheet publicly (Anyone with the link can view)
  5. Enter your Sheet ID and API key in the configuration panel
  6. Test the connection to verify it works

Google Sheet Format: Column A = Room Code (A, B, C, D), Column B = Room Name, Column C = Price
